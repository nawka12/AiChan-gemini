# AiChan-gemini
AiChanBot, but powered by Google Gemini

### This bot is still under construction and lacking a lot of features.

Dependencies:
- Node.js 18 or up
- npm
- @google/generative-ai
- Discord.js
- dotenv

## Setup
- Clone repo
- `npm install`
-  Set `API_KEY` and `DISCORD_TOKEN` on the `.env` file

## Usage

- `ai! [your question here]`
